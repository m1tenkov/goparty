# ГЛАВА 5. ЭКОНОМИЧЕСКАЯ И СОЦИАЛЬНАЯ ОЦЕНКА

## 5.1. Экономическая оценка разработки

Разработка ВК-бота для знакомств среди геймеров «ГоПати» осуществлялась автором самостоятельно в период с февраля 2026 года по май 2026 года. В рамках проекта были выполнены анализ предметной области, проектирование архитектуры системы и базы данных, разработка серверной части, интеграция с API социальной сети «ВКонтакте», настройка удаленного сервера и тестирование работоспособности приложения.

Расчет себестоимости разработки проводился по следующим направлениям:

Трудозатраты автора:

Общее время разработки составило ориентировочно 650 часов, включая:

1. Анализ предметной области, изучение аналогов и формирование требований – 90 часов;

2. Проектирование архитектуры системы и структуры базы данных – 120 часов;

3. Разработка серверной части ВК-бота на Python – 220 часов;

4. Реализация логики регистрации, анкет, лайков, мэтчей и фильтрации – 120 часов;

5. Настройка удалённого сервера, домена, HTTPS и интеграции с ВК API – 40 часов;

6. Тестирование, отладка и доработка пользовательских сценариев – 60 часов;

При условной средней ставке Python-разработчика 700 руб./час потенциальная стоимость трудозатрат составила бы 455 000 рублей. Данная ставка принята на основании исследования заработных плат IT-специалистов, опубликованного сервисом «Хабр Карьера» [23], и используется в качестве условной оценки стоимости работ при коммерческой разработке аналогичного программного продукта.

Фактически разработка выполнялась автором самостоятельно в рамках выпускной квалификационной работы и не требовала прямых затрат на оплату труда. Итоговая калькуляция себестоимости разработки представлена в таблице 7.

Используемое программное обеспечение:

В процессе разработки использовались преимущественно бесплатные программные средства и технологии:

Python — язык программирования серверной части

Visual Studio Code — среда разработки

DBeaver — инструмент администрирования базы данных

XAMPP и phpMyAdmin — локальная работа с MySQL

GitHub — хранение исходного кода и контроль версий

FastAPI — серверный фреймворк

nginx — reverse proxy и веб-сервер

Общие затраты на программное обеспечение составили: 0 рублей.

Облачная инфраструктура и хостинг:

Для размещения серверной части проекта использовалась облачная платформа Cloud.ru Evolution. В рамках проекта был арендован виртуальный сервер стоимостью 150 рублей в месяц. За период разработки и тестирования расходы составили 450 рублей, поскольку в первое время бот разрабатывался на локальном сервере.

Дополнительно был зарегистрирован домен «GoPartybot.ru» через сайт REG.RU стоимостью 169 рублей в год.

Дизайн и проектирование интерфейса:

Проектирование пользовательских сценариев, интерфейсов и диаграмм выполнялось автором самостоятельно с использованием бесплатных инструментов. Дополнительные расходы на UI/UX-дизайн отсутствовали.
