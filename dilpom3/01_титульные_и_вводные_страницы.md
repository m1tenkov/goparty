AННОТАЦИЯ

В данной выпускной квалификационной работе представлена разработка ВК-бота для знакомств среди геймеров «ГоПати». Проект ориентирован на решение проблемы поиска игровых напарников, о которых пользователь заранее знает недостаточно: в большинстве игр отсутствует удобный поиск людей из своего города, а также информация о возрасте, внешности и возможности продолжить общение вне игрового процесса. В рамках работы реализованы серверная часть на Python, интеграция с API социальной сети «ВКонтакте» и база данных MySQL для хранения анкет пользователей, игровых предпочтений, фотографий и истории взаимодействий. Бот позволяет заполнять анкеты, выбирать интересующие игры, просматривать подходящие профили, ставить лайки, отправлять сообщения и получать взаимные совпадения. Также реализованы механизмы модерации, логирования и восстановления состояния диалога после перезапуска. Результатом работы стал функциональный proof of concept специализированного сервиса, адаптированного под российскую аудиторию и реализованного на платформе «ВКонтакте» как доступной альтернативной площадке.

Ключевые слова: ВК-бот, Python, MySQL, бот знакомств, геймеры, анкеты, лайки, мэтчи, ГоПати.

Abstract

This graduation qualification work presents the development of a VK dating bot for gamers called «GoParty». The project is aimed at solving the problem of finding gaming teammates about whom the user initially knows very little: in most games, there is no convenient way to search for people from the same city, as well as obtain information about their age, appearance, or the possibility of continuing communication outside the gaming process. As part of the work, a server-side application in Python, integration with the VK social network API, and a MySQL database for storing user profiles, gaming preferences, photos, and interaction history were implemented. The bot allows users to fill out profiles, select games of interest, browse suitable profiles, like other users, send messages, and receive mutual matches. In addition, mechanisms for moderation, logging, and restoring the dialogue state after a restart were implemented. The result of the work is a functional proof of concept of a specialized service adapted for the Russian audience and implemented on the VK platform as an accessible alternative solution.

Keywords: VK bot, Python, MySQL, dating bot, gamers, profiles, likes, matches, GoParty.
